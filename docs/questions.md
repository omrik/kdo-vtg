# Questions for User

## Features

### Thumbnails
1. Should thumbnails be generated during scan or on-demand?
2. What size should thumbnails be? (e.g., 320x180, 640x360)
3. Should we use a specific frame (10%, middle, keyframe)?

### Advanced Filters
1. What filters do you use most? (resolution, duration, camera type, date)
2. Should filters be combinable (AND/OR logic)?
3. Should saved filter presets be supported?

### Custom Tags
1. Should tags be free-form text or from a predefined list?
2. Who can add tags? (all users or admins only)
3. Should tags be importable/exportable?

### Grid View
1. Should grid show thumbnails or file icons?
2. What info should appear on hover?
3. Should grid be sortable?

## UX/UI

1. Should we support dark/light theme?
2. Do you want keyboard shortcuts for common actions?
3. Should the app be responsive for mobile/tablet?

## Performance

1. How many videos do you typically scan at once? (100, 1000, 10000+)
2. Should we add pagination for large video libraries?

## Integration

1. What NLE software do you use? (Premiere Pro, DaVinci Resolve, Final Cut)
2. Would you like EDL (Edit Decision List) export?
3. Any specific export format needs?

## General

1. Any specific workflow that isn't covered in the user stories?
2. Any pain points with current implementation?
